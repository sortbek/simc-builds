# SimC Builds

Automated SimulationCraft builds for all platforms.

## Releases

Pre-built simc binaries are available on the [Releases](../../releases) page.

| Platform | Architecture | Format |
|----------|-------------|--------|
| Windows | x64 | `.zip` |
| Linux | x64 | `.tar.gz` |
| macOS | ARM64 (Apple Silicon) | `.tar.gz` |

## Schedule

- **Nightly** — Every day at 00:00 UTC
- **Weekly** — Every Monday at 00:00 UTC

Nightly builds are marked as pre-releases. Weekly builds are full releases.

Both workflows can also be triggered manually via the Actions tab.

## Source

All builds are compiled from the latest commit on [simulationcraft/simc](https://github.com/simulationcraft/simc).
